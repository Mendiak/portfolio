01 SOCRATIKA Next.js · React · TypeScript · Firebase

Socratika: Chat With History's Greatest Minds

A full-stack digital Agora that brings history's greatest minds to life through AI conversation. Built with Next.js 15, TypeScript, and Google Genkit.

Features multilingual support, user accounts, Stripe subscriptions, real-time chat via Firebase, and a comprehensive admin interface. Styled with Tailwind CSS and shadcn/ui.

02 CONFESSIO Next.js · React · TypeScript · Firebase

Confessio: AI Digital Sanctuary

A quiet digital sanctuary for anonymous reflection and AI-guided spiritual support. Built with Next.js 15, TypeScript, and Google Genkit.

Features multilingual support, smart content categorization, an interactive Wall of Intentions, and Firebase-powered data persistence. Styled with Tailwind CSS and Radix UI.

03 BITCOIN D3.js · React · TypeScript

Bitcoin History

An interactive D3.js visualization chronicling Bitcoin's price history with key events. Features logarithmic/linear scaling, zoom, event markers, and light/dark themes.

Built with Vite, D3.js v7, Bootstrap 5.3, and Tailwind CSS. Supports multiple languages and responsive layouts for exploring Bitcoin's financial journey.

04 LAMBO Vanilla JS · CoinDesk API

When Lambo

A humorous Bitcoin-to-Lamborghini calculator that tells you how close you are to owning a supercar based on your crypto holdings.

Features real-time CoinDesk API data, multiple fiat currencies, confetti celebrations, and customizable Lamborghini models. Built with vanilla JS and Google Fonts (Cormorant, Proza Libre).

05 PULSEROOTS D3.js · HTML5 · CSS3 · Spotify API

PulseRoots: Interactive Electronic Music Genre Explorer

An interactive visualization of electronic music's evolution. Built with Astro and D3.js v6, it maps the complex web of genres and subgenres into an explorable hierarchical tree.

Features detailed genre info panels, embedded Spotify previews, and responsive design. Explore major families like Ambient, Techno, and House and their subgenres.

07 DRAKE Vanilla JS · HTML5 · CSS3

Drake Equation Visualizer

An interactive 3D visualization of the Drake Equation that estimates the number of extraterrestrial civilizations in the Milky Way. Built with Three.js, Chart.js, and vanilla JavaScript.

Adjust each parameter with real-time sliders and watch how it influences the final estimate. Features a 3D starfield, interactive charts, and a clean responsive layout.

View Project Open Source
08 PLAINTEXT Vanilla JS · Canvas · Google Fonts

Plaintext: Typography-Driven Wallpaper Generator

A vanilla JavaScript and HTML5 Canvas wallpaper generator that transforms curated quotes into high-resolution typographic posters with Swiss design-inspired layouts.

Features a bilingual interface (English/Spanish), dark mode, Zen mode, Google Fonts integration, custom color controls, and export options.

View Project Open Source
09 ONLY 3 LEFT Next.js · React · TypeScript · Tailwind CSS

Only 3 Left: Deceptive UX Patterns

An interactive catalogue of deceptive UX patterns — dark patterns, manipulative interfaces, and persuasive mechanisms used to influence decisions online. Built with Next.js 16, TypeScript, and Tailwind CSS.

Features 40 documented patterns across 10 categories, interactive live specimens (fake countdowns, cookie simulators, pricing traps), severity ratings, psychological breakdowns, and ethical alternatives. Fully bilingual (EN/ES).

View Project Open Source
